# DevTrack

DevTrack is a developer productivity tracker built with React, TypeScript, Vite, and Tailwind CSS. Log your daily progress, visualize insights, and showcase your growth.

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
